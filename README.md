## 1. Název projektu
**Orientační běh – analýza a databázové zpracování**  

## 2. Popis projektu
Tato ročníková práce se věnuje sportu zvanému orientační běh (OB), který spojuje fyzickou zdatnost s navigačními schopnostmi. Cílem práce je nejen představit principy a pravidla tohoto sportu, ale také analyzovat data účastníků, mapových bodů a výsledků závodů pomocí konceptuálního modelu a databázového návrhu.  

Práce obsahuje:
- Přehled orientačního běhu a jeho disciplín  
- Analýzu dat účastníků, map a tratí  
- E-R diagram znázorňující vztahy mezi jednotlivými entitami  
- Návrh databázového schématu  
- Doporučení pro případné nasazení nebo implementaci systému  

## 3. Obsah
1. **Úvod** – seznámení s orientačním během, historie a význam sportu  
2. **Analytická část** – popis entit, dat a vztahů, E-R diagram  
3. **Návrhová část** – návrh databáze, možné implementace, user flow diagram  
4. **Závěr** – shrnutí výsledků a doporučení pro další rozvoj  

## 4. Použití
- Projekt slouží primárně jako dokumentace a analytický podklad pro orientační běh.  
- Diagramy lze využít pro implementaci databázového systému, který eviduje účastníky a výsledky závodů.  
- SQL skripty obsahují příklady, jak lze data organizovat a dotazovat.  

## 5. Požadavky
- Přístup k nástroji pro zobrazení diagramů (např. draw.io, Lucidchart)  
- Možnost otevřít textové dokumenty (.docx, .pdf)  
- (Volitelné) SQL server nebo databázový nástroj pro testování návrhu databáze
